Student Management System

A desktop-based Student Management System developed using Python and the Tkinter library. This application helps manage student records efficiently through a simple and user-friendly graphical interface.

📌 Features

Add new student records
View all student details
Search student records
Update existing student information
Delete student records
User-friendly GUI using Tkinter
Data storage using SQLite database
Input validation for accurate data entry

🛠️ Technologies Used

Python 3.x
Tkinter (GUI Framework)
SQLite3 (Database)

Installation:
Clone the repository:
git clone <repository-url>
Navigate to the project directory:
cd Student-Management-System
Run the application:
python main.py

Usage:
Launch the application.
Enter student details such as:
Student ID
Name
Age
Department
Contact Number
Click Add Student to save records.
Use Search, Update, or Delete options to manage records.
View all student information in the records table.

Learning Outcomes:

Through this project, you will gain experience in:

Python programming
GUI development using Tkinter
Database integration with SQLite
CRUD operations (Create, Read, Update, Delete)
Event-driven programming
